# Excel Sales Data Analysis Project

## Overview
This project demonstrates hands-on experience with advanced Excel features
used for structured data analysis and automation-ready workflows.

## Dataset
- Retail sales dataset (5,000+ rows)
- Columns include Date, Product, Region, Revenue, Quantity

## Excel Features Used
- PivotTables and PivotCharts
- Advanced formulas (IF, VLOOKUP / XLOOKUP)
- Data validation for controlled inputs
- Conditional formatting for insights
- Structured Excel tables

## Workflow
1. Cleaned raw data and removed inconsistencies
2. Created PivotTables to analyze sales by region and product
3. Built dynamic charts for trend analysis
4. Applied data validation and formatting
5. Documented each change step-by-step

## Documentation
- Each spreadsheet modification follows a structured workflow
- Changes were documented to ensure reproducibility

## Tools
- Microsoft Excel
